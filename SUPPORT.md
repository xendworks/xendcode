## Reporting Issues

2. **XendCode version**: Check in Extensions view
3. **Operating System**: Windows/macOS/Linux
4. **Steps to reproduce**: Clear description of what you did
5. **Expected behavior**: What should happen
6. **Actual behavior**: What actually happened
7. **Error messages**: Any error messages or logs

We welcome feature requests! Please submit them as GitHub Issues with:

- Any examples or mockups if applicable
